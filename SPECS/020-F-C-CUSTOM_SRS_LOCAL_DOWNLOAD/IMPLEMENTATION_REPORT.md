# Отчёт по реализации: Локальное скачивание пользовательских правил «SRS»

## 1. Статус

- Статус задачи: **Complete**
- Кодовая реализация: **выполнена**

## 2. Краткое содержание

- Добавлена поддержка локального скачивания SRS для пользовательских правил типа **SRS** (Wizard → Rules tab):
  - кнопка `⬇/🔄/✔️` для custom SRS-правил;
  - включение чекбокса запускает скачивание при отсутствии локальных SRS;
  - при наличии `bin/rule-sets/<tag>.srs` генерация конфига использует `type: "local"` с `path` вместо remote `url`.

## 3. Изменённые файлы

- `core/services/srs_downloader.go` — общие helper-ы для SRS (в т.ч. группы скачивания и GitHub-частный случай).
- `ui/wizard/business/create_config.go` — использование `local` rule_set для пользовательских SRS при наличии локального файла.
- `ui/wizard/tabs/rules_tab.go` — UI/логика кнопки SRS и скачивания для custom SRS-правил.
- `docs/release_notes/upcoming.md` — пункт релизных заметок.
