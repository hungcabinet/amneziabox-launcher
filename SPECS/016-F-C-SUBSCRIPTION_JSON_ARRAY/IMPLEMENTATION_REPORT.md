## Причина закрытия

Реализация по **SPEC/PLAN/TASKS** не велась: в коде нет `ParseNodesFromJSONArrayConfigs`, декодер подписок по-прежнему возвращает ошибку для ответов, начинающихся с `[` (см. `core/config/subscription/decoder.go`). Задача закрывается как завершённая единица планирования без кода, чтобы зафиксировать факт и убрать статус «в плане навечно».
